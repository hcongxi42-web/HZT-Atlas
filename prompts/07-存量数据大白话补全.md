# 存量数据大白话补全提示词
## 使用方式
1. 将已有产业数据文件的完整代码粘贴给 AI
2. AI 会为每个节点生成 `plainLanguageDescription`
3. 输出可直接替换原文件的完整代码

---

## 提示词（复制使用）
```
请为以下产业链数据中的每个节点补全 "plainLanguageDescription" 字段（大白话解释）。

## 输入数据

```typescript
【粘贴当前产业数据文件的完整代码】
```

## 任务要求

### 目标
1.为产业链写一个介绍+大白话解释，注意要解释每一个节点的用处和重要性，而不是仅仅列出节点名称。注意逻辑通顺
2.为每个节点（rootNodes 中的根节点 和 childNodes 中的所有子节点）补充 `plainLanguageDescription` 字段。
3.如果节点已有 `plainLanguageDescription`，请检查其质量，如不达标则重写。

### 大白话解释写作规范（必须严格遵守）
大白话或者补充解释可以使用小字号注释，保证清晰可见。
1. **专业缩写规范**：
   - EDA → "芯片设计软件（EDA）"
   - IP核 → "芯片里的功能模块设计图（IP核）"
   - IGBT → "电动车里控制电流的'开关'（IGBT）"
   - HBM → "让AI算得更快的高带宽存储芯片（HBM）"

2. *通俗比喻**：
   - 光刻机 → "芯片制造的照相机，把电路图'拍'到硅片上"
   - 电解液 → "电池里的'血液'，让锂离子在正负极之间跑起来"
   注意：可以写成： 光刻机，然后附上小字解释： 光刻机，把电路图'拍'到硅片上
3. **说明价值**（不是"这是什么"，而是"这有什么用、为什么重要"）：
   - ❌ "磷酸铁锂是一种正极材料"
   - ✅ "电动车电池里最常用的正极材料，成本低、安全性高，决定了电池能充多少次电"

4. **关联熟悉事物**：
   - 变压器 → "电力系统的'血压调节器'，把高压电变成家用220V"
   - 智能电表 → "家里电表升级成能联网的版本，供电公司能远程抄表，用户能看实时用电量"

5. **一句话长度**：控制在 30-60 字，节点卡片上能直接显示

6. **严禁行为**：
   - ❌ 复制粘贴 description 的内容
   - ❌ 使用"某某领域的重要环节"这类空话
   - ❌ 出现英文缩写未解释
   - ❌ 过于冗长（超过80字）

### 输出格式

输出**完整的更新后文件**（可直接替换原文件）。

只修改/新增 `plainLanguageDescription` 字段，其他所有内容保持原样不动。

```typescript
// === 补全后的完整文件 ===
[完整的 TypeScript 代码]
```

同时输出补全摘要：

```markdown
## 大白话补全摘要

【产业名称】

- 根节点：N个，补全N个
- 子节点：N个，补全N个
- 总计：N个节点补全了 plainLanguageDescription

### 示例（展示3-5个典型节点）

| 节点名称 | 大白话解释 |
|---------|-----------|
| ... | ... |
```
```
